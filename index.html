Webinar Overview

Operators looking to build scalable, flexible and reliable data centers are moving to layer 3 Clos architectures. To be truly web scale, a Clos design needs to be coupled with standardized hardware, an open network OS and various software tools. Together, they enable operators to achieve operational efficiency and economical scalability.

In this webinar we discuss the architecture(s) of a BGP layer 3 design along with its benefits. During the session, we examine how Cumulus Networks, Dell and Red Hat simplified a 300+ node OpenStack deployment and built a highly scalable architecture in under 6 hours.

Join us and learn:

  • Benefits of BGP layer 3 architecture

  • Scalability via NetDevOps, leveraging automation tools like Ansible across the stack

  • Simplicity and reliability with Cumulus Routing on the Host
